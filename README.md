# 🛡️ SmartFaceGuard

> AI-Powered Identity Verification & Face Authenticity Verification System

SmartFaceGuard is an AI-powered face authenticity verification platform that enhances traditional biometric authentication by combining liveness detection, deepfake detection, replay attack detection, and image quality assessment into a single intelligent verification pipeline.

Unlike traditional face recognition systems that only answer **"Who is this person?"**, SmartFaceGuard first answers **"Can this face be trusted?"**

---

# 🚀 Problem Statement

Modern authentication systems are vulnerable to:

- AI Generated Faces
- Deepfake Videos
- Replay Attacks
- OBS Virtual Camera
- High Resolution Screen Playback

Most biometric systems only verify identity.

SmartFaceGuard introduces an additional AI-powered trust layer before identity verification.

# 💻 Installation

## Clone Repository

```bash
git clone https://github.com/AlphaMALE888/SmartFaceGuard.git
```

---

## Backend

```bash
cd backend

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
